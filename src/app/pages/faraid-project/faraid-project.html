Final Year Project Case Study

Faraid A-Z Mobile Application

Educational Mobile Application for Learning and Calculating Islamic Inheritance

Faraid A-Z is a mobile application developed as my Final Year Project to help users learn, understand, and calculate Islamic inheritance distribution in a clearer and more accessible way.

Flutter Dart MySQL GitHub

Overview

Making faraid learning clearer and easier to access.

The application combines educational content, guided references, and a calculator flow so users can learn Islamic inheritance concepts and better understand how distribution is determined.

Development Process

From research to usable mobile prototype.

  1. Studied user needs and faraid learning pain points.
  2. Planned app modules for learning content, glossary, FAQ, and calculation.
  3. Designed mobile screens with clear navigation and readable content sections.
  4. Implemented the core application using Flutter, Dart, and supporting data storage.
  5. Evaluated usability and user acceptance with participant feedback.

Challenges

Balancing accuracy, clarity, and usability.

The main challenge was presenting a complex Islamic inheritance topic in a way that remains understandable for general users while keeping the interface simple, organized, and suitable for mobile screens.
